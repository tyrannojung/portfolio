[ABC] MOZAC

요약

  • 카루스웍스 x ABC
  • 모두가 참여하는 작품 MOZAC - 블록체인 기반 웹소설 프로젝트(백서 : 링크)
  • 테스트 홈페이지 : https://mcop.nodehome.io/

역할

  • 프로젝트 전체 인원 : 5명
  • 나의 역할과 기여도 : 블록체인, 퍼블리싱 제외한 모든 기능 및 테스트 1인 구현

성과

  • 사내 블록체인 API를 활용하여 1인 웹개발(Spring)을 성공적으로 마무리
  • TTA 소프트웨어시험인증 6가지 항목 성공적으로 통과 후 인증

시기

  • 프로젝트 진행 기간 (2021.8 ~ 2021.11)

📝TTA 시험인증

인증과정

  • 회사 블록체인 API 테스트 진행
    • 본인이 직접 apache jmeter와 리눅스 쉘스크립트를 활용하여 코드 작성
    • 프로젝트와 블록체인 테스트 본인이 직접 TTA(상암)에 가서 시연 및 테스트 진행

인증항목

  • 초당 트랜잭션 처리용량
    • 블록체인 1개 노드에서 초당 트랜잭션 처리 건수를 측정 특정 계정의 토큰 잔고를 10,000 번 조회하여 초당 처리속도 (TPS) 측정한다.
    • 목표결과 : 1000 TPS 이상
  • 트랜잭션 및 블록 송수신 시간
    • 트랜잭션 및 블록 송수신 시간을 측정 블록체인에 등록된 컨텐츠 데이터에 대하여 논리식에 의한 검색 조건으로 검색을 시도하고, 100건이상의 동시 검색에 대한 시간을 측정
    • 목표결과 : 5 초이내
  • 트랜잭션 처리 시간
    • 개별 트랜잭션이 검증되고 승인되는 시간을 측정 블록체인에 100개의 컨텐츠 데이터를 등록 하고 트랜잭션 완료 시간을 확인한다.
    • 목표결과 : 평균 건당 1초 이내 ( 전체 소요시간 / 등록 건수)
  • 블록체인 기반 플랫폼 동시접속자 수
    • 45명 이상이 동시에 플랫폼의 서비스에 접속하여 정상 동작 여부 확인
    • 목표결과 : 1000 TPS 이상
  • 거버넌스 체계 구현 종류
    • 플랫폼에 구현된 거버넌스 체계가 시스템에 반영되어 동작되는 것을 측정
    • 목표결과 : 정상적으로 보상 정책이 적용됨
  • 블록체인 키 값과 MOZAC 사용자 정보 연동
    • 블록체인 키 값과 MOCP 사용자 정보가 연동되는 트랜잭션을 블록에서 확인
    • 목표결과 : 정상 동작
