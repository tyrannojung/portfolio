[ABC] HeavenArt - NFT Market Place

요약

  • NFT Market Place
  • 다운로드 : https://play.google.com/store/apps/details?id=io.nodehome.ha.android.haapp

역할

  • 프로젝트 전체 인원 : 10명
  • 나의 역할 : 스프링 백엔드 개발, 블록체인 API연동, IPFS,채팅개발, 서버설치
  • 프로젝트 기여도 : 30%

성과

  • 회원(가입, 수정, 복원 등), NFT : 생성, 판매(경매, 즉시구매, 가격제안), 상세, 전송등 개발완료
  • IPFS노드, 채팅서버를 구축하고 docker를 통한 모듈화 완료

시기

  • 프로젝트 진행 기간 (2022.3 ~ 2022.6)

📝HeavenArt

  • 회사 솔루션 hyperledger fabric기반nodeohmeMainnet체인을 활용한 NFT Market Place
  • NFT생성(erc721, 1155) 및 판매가능
  • NFT를 외부거래소(opensea,rarible 등)에 전송 가능
  • 플랫폼 내부에 내장 지갑 존재, Token 생성 및 스왑, 전송가능
  • 블록체인 종합 플랫폼

📝개발

NFT

  • 회사에서 개발한 hyperledger fabric블록체인 API를 활용하여 개발
  • Mint, 조회, 판매 등의 NFT기능을 Spring과 Javascript를 통해 개발

IPFS

  • 회사내에 사용해본사람이 전무하여 처음부터 연구
  • ipfs/go-ipfs

  • IPFS노드와 연동하는 API제작
    1. IPFS ADD

      Curl

      curl -X 'POST' \ 'https://ipfs.nodehome.io/ipfs' \ -H 'Content-Type: multipart/form-data' \ -F ipfs_file=@"/sample-result.json”

      Response body

      {"result": "string","value": Array}

      Example Value

      { "result":"OK", "value":["Qmbw24HyDYvAgs3cQn3Rap3V6v1LFCraAVrveqAEyUAYAU","Qmf7Pz4aTqiHmpmxmePiSsdfm9dJTqFTDZLv5A8sjR6JyM"]}

  1. IPFS GET

    Curl

    curl -X 'GET' \ 'https://ipfs.nodehome.io/ipfs/${IPFS_YOUR_HASH}' \

    Response

    On success, the call to this endpoint will return with 200
