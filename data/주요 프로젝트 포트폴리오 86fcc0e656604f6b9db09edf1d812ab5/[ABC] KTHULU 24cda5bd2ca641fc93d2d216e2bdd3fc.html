📝프로젝트 개발

프로젝트 1인개발

  • Manifest v3 버전 확장 브라우저 개발
  • 고객 데이터는 오직 로컬에 있으며, 서버로 보내지 않는 탈중앙화 구조 구현
  • 로컬 고객 데이터를 web3 연동으로 잔고 조회 기능 구현
  • 이더리움 클레이튼 폴리곤 솔라나를 연구하여 endpoint를 활용한 차별화 구현
  • Kthulu RPC라는 명칭의 API를 스프링으로 제작 (ex infura.io사용 방법과 비슷)
  • Extension 호출 API 자바스크립트로 제작 완료 및 효성그룹에 제공 완료
